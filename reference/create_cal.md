# create cal

create cal

## Usage

``` r
create_cal(df, file = NULL)
```

## Arguments

- df:

  a semester tibble with name and date

- file:

  ICS file to export cal as

## Value

string for ics

## Examples

``` r
df <- create_semester(lubridate::today(), weeks = 8, MSB = 4)
create_cal(df)
```
