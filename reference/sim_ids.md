# Simulate multiple IDs

Simulate multiple IDs

## Usage

``` r
sim_ids(n, n_digits = 7)
```

## Arguments

- n:

  number of IDs

- n_digits:

  number of digits in each ID

## Value

vector of IDs

## Examples

``` r
sim_ids(10)
#>  [1] "a4265346" "a2953254" "a3849174" "a8879993" "a9317594" "a6873838"
#>  [7] "a8272152" "a9917419" "a9462225" "a3425563"
```
